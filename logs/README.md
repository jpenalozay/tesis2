# `logs/`

Registro de la ejecución de pipelines. La ingesta v0 escribe además a consola; **archivo** típico: `ingesta.log` (regenerable, excluido de Git; ver `.gitignore`).

Otras salidas (métricas, resúmenes) pueden añadirse siguiendo el mismo criterio que en [EjemploProyecto](https://github.com/GlenRodriguez/EjemploProyecto).
