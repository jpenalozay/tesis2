### Información Clave

- **Estado**: Fase 1 completada al 85%
- **Próximo hito**: Implementar gRPC Python ↔ Go
- **Decisión clave**: Gitea > MinIO (ver PROYECTO_COMPLETO.md sección 3.2)
- **Arquitectura**: Python (50%) + Go (50%), NO Rust

### Comandos para Empezar

```bash
# 1. Levantar infraestructura
cd docker
docker-compose -f docker-compose.infra.yml up -d

# 2. Compilar Go
go build -o bin/framework go/cmd/main.go

# 3. Ejecutar
./bin/framework
```
