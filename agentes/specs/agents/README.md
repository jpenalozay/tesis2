## 🤖 AGENTES DISPONIBLES

### ✅ Agente DB (Database)

**Archivos Disponibles**:
- `db_agent.json` - ⚙️ Configuración para Redis/comunicación entre agentes
- `db_agent.md` - 📖 Guía completa para desarrolladores (lectura humana)
- `db_agent_knowledge.json` - 🧠 Base de conocimiento del modelo de datos (para análisis inteligente)
- `db_agent_knowledge.md` - 📚 Guía sobre el conocimiento y sugerencias inteligentes
- `db_agent_test_performance.json` - 📊 Configuración de pruebas y análisis de performance
- `db_agent_test_performance.md` - 📈 Guía de pruebas y performance automatizadas
- `AGENTE_DB_ESPECIFICACION_MEJORADA.md` - 📋 Especificación técnica detallada

**Estado**: ✅ Especificación completa y mejorada

**Contenido**:
- Funciones principales con niveles de importancia
- Reglas CRITICAL, HIGH, MEDIUM, LOW
- Naming conventions completas
- Base de conocimiento del modelo actual
- Sistema de sugerencias inteligentes
- 📊 Plan de pruebas automatizadas (diarias y semanales)
- ⚡ Plan de análisis de performance (horario, diario, semanal)
- 🔍 Detección automática de cuellos de botella
- 🧐 Investigación automática de problemas de performance
- 💡 Sugerencias de optimización basadas en volumen
- Adaptado para entorno de desarrollo
- Ejemplos prácticos

**Funciones principales**:
- `validate_model()` - Validación completa de modelos
- `check_migrations()` - Validación de migraciones SQL
- `optimize_queries()` - Optimización de queries
- `validate_connection_pool()` - Validación de pool
- `validate_naming_conventions()` - Validación de naming
- `analyze_and_suggest()` - 🆕 Análisis inteligente y sugerencias basadas en conocimiento
- `update_knowledge_base()` - 🆕 Actualización automática del conocimiento
- `run_tests()` - 🆕 Ejecución de pruebas automatizadas
- `analyze_performance()` - 🆕 Análisis de performance por volumen
- `detect_bottlenecks()` - 🆕 Detección de cuellos de botella
- `investigate_performance_issues()` - 🆕 Investigación de problemas de performance

**Para leer**:
- 👤 **Desarrolladores**: Lee `db_agent.md` para entender cómo funciona
- 🧠 **Conocimiento y Sugerencias**: Lee `db_agent_knowledge.md` para entender el sistema de análisis inteligente
- 📊 **Pruebas y Performance**: Lee `db_agent_test_performance.md` para entender el sistema de pruebas y performance
- 🤖 **Sistema/Redis**: Usa `db_agent.json` para configuración y comunicación
- 📊 **Conocimiento del Modelo**: Usa `db_agent_knowledge.json` para análisis automático
- ⚙️ **Configuración de Pruebas**: Usa `db_agent_test_performance.json` para configurar pruebas y análisis
- 🔧 **Implementación**: Consulta `AGENTE_DB_ESPECIFICACION_MEJORADA.md` para detalles técnicos
